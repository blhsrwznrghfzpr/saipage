# Saipage

A diceroll tool for TRPG


## Official site

https://ysakasin.github.io/saipage/


## Build Requires

- Node (^10.0.0)


## Build

``` bash
# install dependencies
npm install

# build client
npm build
```

## Run dev-server

```
npm start
```
