# Saibox

A TRPG session tool

## Requires

- Node (>= 8)
- MongoDB


## Build

``` bash
# install dependencies
npm install

# build client and start web server
npm run start
```

## Configure

Saibox are configured by environment variables.

### `PORT`

Port of web server

default: `8080`

### `MONGO_URI`

URI of connecting mongodb

default: `mongodb://localhost:27017`
